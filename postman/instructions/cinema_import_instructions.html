Cinema API Postman Collection Import Instructions

Step 1: Download Postman

If you don't have Postman installed, download and install it from https://www.postman.com/downloads/

Step 2: Import the Collection

  1. Open Postman
  2. Click the "Import" button in the top left corner
  3. Select the "Files" tab
  4. Click "Upload Files" and select the Cinema_API_Collection.json file
  5. Click "Import"

Step 3: Import the Environment

  1. In Postman, click the "Import" button again
  2. Select the "Files" tab
  3. Click "Upload Files" and select the Cinema_API_Environment.json file
  4. Click "Import"

Step 4: Select the Environment

  1. In the top right corner of Postman, click the environment dropdown
  2. Select "Cinema API Environment" from the list

Step 5: Get Cinema Data

  1. In the collection, expand the "Cinemas" folder
  2. Click on the "Get All Cinemas" request
  3. Click "Send" to execute the request
  4. From the response, copy a cinema ID
  5. Click the "Environment" quick look button (eye icon) in the top right
  6. Set the "cinemaId" variable to the copied ID

Step 6: Authenticate as Admin

  1. Expand the "Authentication" folder
  2. Click on the "Login Admin" request
  3. Modify the email and password in the request body if needed
  4. Click "Send" to execute the request
  5. From the response, copy the token value
  6. Set the "adminToken" environment variable to the copied token

Step 7: Test Cinema Operations

  1. Now you can test various cinema operations:
  2. Get cinema details using "Get Cinema by ID"
  3. Create a new cinema using "Create Cinema (Admin)"
  4. Add a room to a cinema using "Add Room to Cinema (Admin)"
  5. Create screenings for movies at the cinema

Note:

Admin operations require authentication with a valid JWT token. Make sure you've completed Step 6 before testing these endpoints.

When creating a new cinema or room, make sure to review the request body and modify it as needed for your testing scenario.
