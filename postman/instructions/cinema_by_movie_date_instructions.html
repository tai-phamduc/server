Cinema By Movie and Date - Postman Collection Guide

Step 1: Download Postman

If you don't have Postman installed, download and install it from https://www.postman.com/downloads/

Step 2: Import the Collection

  1. Open Postman
  2. Click the "Import" button in the top left corner
  3. Select the "Files" tab
  4. Click "Upload Files" and select the Cinema_By_Movie_Date_Collection.json file
  5. Click "Import"

Step 3: Import the Environment

  1. In Postman, click the "Import" button again
  2. Select the "Files" tab
  3. Click "Upload Files" and select the Cinema_By_Movie_Date_Environment.json file
  4. Click "Import"

Step 4: Select the Environment

  1. In the top right corner of Postman, click the environment dropdown
  2. Select "Cinema By Movie Date Environment" from the list

Step 5: Set Up Date Variables

You need to set up the date variables for today and tomorrow. You can do this in two ways:

Option 1: Manual Setup

  1. Click the "Environment quick look" button (eye icon) in the top right
  2. Set the todayDate variable to today's date in YYYY-MM-DD format (e.g., "2023-12-01")
  3. Set the tomorrowDate variable to tomorrow's date in YYYY-MM-DD format (e.g., "2023-12-02")
  4. Click outside the environment panel to close it

Option 2: Using the Script

  1. Click on the collection name "Cinema By Movie and Date Collection" in the sidebar
  2. Click on the "Pre-request Scripts" tab
  3. Copy and paste the following script:
// Get today's date in YYYY-MM-DD format
const today = new Date();
const todayFormatted = today.toISOString().split('T')[0];

// Get tomorrow's date in YYYY-MM-DD format
const tomorrow = new Date();
tomorrow.setDate(tomorrow.getDate() + 1);
const tomorrowFormatted = tomorrow.toISOString().split('T')[0];

// Set environment variables
pm.environment.set('todayDate', todayFormatted);
pm.environment.set('tomorrowDate', tomorrowFormatted);

console.log('Date variables set:');
console.log('todayDate:', todayFormatted);
console.log('tomorrowDate:', tomorrowFormatted);
  • Click "Save"
  • Run any request in the collection to trigger the script

    • Step 6: Find a Movie ID

    1. Expand the "Movies" folder in the collection
    2. Click on "Get Now Playing Movies"
    3. Click "Send" to execute the request
    4. From the response, copy the ID of a movie you want to use for testing
    5. Click the "Environment quick look" button (eye icon) in the top right
    6. Set the movieId variable to the copied ID

    Step 7: Find Cinemas Showing the Movie

    1. Expand the "Cinemas by Movie and Date" folder
    2. Click on "Get Cinemas by Movie and Today's Date"
    3. Click "Send" to execute the request
    4. The response will show all cinemas that are showing the selected movie today
    5. If you get an empty array, try "Get Cinemas by Movie and Date" with a different date, or try a different movie

    Step 8: Get Cinema Details and Screenings

    1. From the response in Step 7, copy a cinema ID
    2. Set the cinemaId variable to the copied ID
    3. Expand the "Cinema Details" folder and run "Get Cinema by ID" to see detailed information about the cinema
    4. Expand the "Screenings" folder and run "Get Screenings by Movie, Cinema and Date" to see all showtimes for the movie at the selected cinema

    Note:

    If you're not getting any results, try the following:

    • Use a different movie ID (some movies might not have any screenings)
    • Try a different date (the movie might not be showing on the selected date)
    • Check that the date format is correct (YYYY-MM-DD)
    • Verify that the API base URL is correct
