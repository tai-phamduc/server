Movie API Postman Collection Import Instructions

Step 1: Download Postman

If you don't have Postman installed, download and install it from https://www.postman.com/downloads/

Step 2: Import the Collection

  1. Open Postman
  2. Click the "Import" button in the top left corner
  3. Select the "Files" tab
  4. Click "Upload Files" and select the Movie_API_Collection.json file
  5. Click "Import"

Step 3: Import the Environment

  1. In Postman, click the "Import" button again
  2. Select the "Files" tab
  3. Click "Upload Files" and select the Movie_API_Environment.json file
  4. Click "Import"

Step 4: Select the Environment

  1. In the top right corner of Postman, click the environment dropdown
  2. Select "Movie API Environment" from the list

Step 5: Authenticate

  1. In the collection, expand the "Authentication" folder
  2. Click on the "Login" request
  3. Click "Send" to execute the request
  4. From the response, copy the token value
  5. Click the "Environment" quick look button (eye icon) in the top right
  6. Set the "userToken" variable to the copied token
  7. If you have admin credentials, repeat the process for the "adminToken" variable

Step 6: Start Testing

You can now use the collection to test various movie-related API endpoints:

  1. Expand the "Movies" folder to see available requests
  2. Click on "Get All Movies" and click "Send" to execute the request
  3. From the response, copy a movie ID and set it as the "movieId" environment variable
  4. Continue testing other endpoints as needed

Note:

Some endpoints require authentication with a valid JWT token. Make sure you've completed Step 5 before testing these endpoints.

Admin endpoints require an admin token. You'll need admin credentials to access these endpoints.
